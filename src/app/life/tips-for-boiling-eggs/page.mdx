## 煮的时候不裂壳的小技巧

| 技巧         | 原理               | 操作步骤                      |
| ------------ | ------------------ | ----------------------------- |
| 室温鸡蛋下锅 | 冷蛋遇热涨裂       | 煮之前从冰箱取出放 10-15 分钟 |
| 冷水下蛋     | 避免骤热           | 把鸡蛋放入冷水再开火          |
| 加盐 or 白醋 | 强化蛋壳、凝固蛋白 | 水中加 1 小勺盐或少量白醋     |
| 小火慢煮     | 气室膨胀更平稳     | 水开后保持中小火煮            |
| 让蛋横躺煮   | 蛋黄居中好看       | 下水前在桌面轻轻滚动几圈      |

## 怎么煮才能更好剥壳

| 技巧               | 为啥好剥               | 怎么做                      |
| ------------------ | ---------------------- | --------------------------- |
| 全熟或“八分熟”最佳 | 蛋白结实不粘壳         | 7-10 分钟左右               |
| 煮好立刻冰水降温   | 收缩脱壳               | 捞出马上泡冰水 1~3 分钟     |
| 从“气室”那头开始敲 | 那里有空气、好起壳     | 蛋尖朝下、从较圆那头敲裂    |
| 水中剥壳           | 水能渗入膜层           | 用流动水/清水碗剥更快更整洁 |
| 老一点的蛋更好剥   | 新蛋 pH 低，蛋白更黏壳 | 流通 3-5 天的蛋最适合       |

## 时间参考（根据熟度）

| 口感   | 时间（开煮后计时） | 描述       |
| ------ | ------------------ | ---------- |
| 溏心   | 6 分钟             | 中间流心   |
| 八分熟 | 7-8 分钟           | 软糯蛋黄   |
| 全熟   | 9-10 分钟          | 坚实不粉化 |

## 完整示范流程（极简版）

1. 鸡蛋从冰箱取出 → 放常温 10-15 分钟
2. 冷水放入锅中 → 加 1 小勺盐或少量白醋
3. 中火加热 → 小滚后保持中小火
4. 根据口感计时 6-10 分钟
5. 捞出立即泡冰水 1-3 分钟
6. 从圆头（气室处）敲裂 → 在水中剥壳
